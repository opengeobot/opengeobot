# OpenGEO Bot V1 安全与合规检查清单

<!--
作者：AxeXie
创建时间：2026-05-08 13:26:00
-->

## 1. 密钥与凭证管理

- [ ] API Key 与 Token 不明文存储。
- [ ] 凭证读取走安全配置通道。
- [ ] 日志中无凭证明文输出。

## 2. 敏感信息保护

- [ ] Prompt、资产内容、日志已接入敏感信息检测。
- [ ] 邮箱、手机号、身份证号等字段已脱敏或阻断。
- [ ] 数据导出流程包含二次脱敏。

## 3. 审计与追溯

- [ ] 关键操作有审计日志（操作者、时间、对象、动作、结果）。
- [ ] traceId 可追溯跨模块调用链路。
- [ ] 审计日志保留策略满足合规要求。

## 4. 引擎调用合规

- [ ] 优先使用官方 API 或官方 SDK。
- [ ] 调用方式、限制和免责声明对用户可见。
- [ ] 不默认启用违规抓取策略。

## 5. 数据生命周期

- [ ] 数据保留期可配置并生效。
- [ ] 项目删除流程可执行并可审计。
- [ ] 回滚与恢复流程已演练。
