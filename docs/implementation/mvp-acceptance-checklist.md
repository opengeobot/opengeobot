# OpenGEO Bot V1 MVP 终验清单

<!--
作者：AxeXie
创建时间：2026-05-08 13:16:00
-->

## 1. 功能验收（对照 PRD 13）

- [ ] 可完成网站或开源仓库接入、提示词配置、基线评测与洞察查看。
- [ ] 可识别提及、引用、位置、情感与竞品差距。
- [ ] 可输出不少于 20 个可解释优化机会。
- [ ] GEO Bot 可产出至少一种可落地交付物（Markdown/改写建议/PR 草稿）。
- [ ] 可执行优化后重跑并对比 Baseline 与 After 指标变化。
- [ ] 可按周自动跟进核心提示词并生成监控报告。
- [ ] 可记录历史 Playbook 与效果结果并用于下一轮推荐。
- [ ] 具备统一数据字典、统一日志、统一系统配置能力并支持国际化输出。

## 2. 非功能验收

- [ ] 可复现性：Run 元数据记录完整。
- [ ] 稳定性：单引擎失败不影响其他引擎任务结果。
- [ ] 性能：满足 1000 提示词/3 引擎/每日监控目标。
- [ ] 可观测性：trace 全链路可检索，审计日志完整。

## 3. 安全与合规验收

- [ ] 密钥与 Token 无明文落盘。
- [ ] 敏感信息检测与脱敏策略生效。
- [ ] 引擎调用符合官方 API 与授权约束。
- [ ] 数据保留与删除策略可执行。
